# RoR2_Japanese_Unofficial
RoR2の非公式日本語翻訳です

## 適用方法

1. **Risk Of Rain 2** の ローカルファイルを閲覧
1. **\Risk of Rain 2_Data\Language** を開き jau フォルダをコピー
1. ゲーム内のメインメニュー右上から言語を **日本語(非公式-XXXXXX)** に設定
1. 楽しんで！
